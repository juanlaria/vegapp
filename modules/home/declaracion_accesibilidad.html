Declaración de accesibilidad de Vegapp

Vegapp S.A. se compromete a garantizar la accesibilidad digital para personas con discapacidad. Estamos continuamente mejorando la experiencia de usuario para todas las personas, y aplicamos criterios de accesibilidad relevantes.

Medidas para apoyar la accesibilidad

Vegapp S.A. tomas las siguientes medidas para garantizar la accesibilidad de Vegapp:

  • Incluir la accesibilidad como parte de nuestra misión.
  • Incluir la accesibilidad a través de políticas internas.
  • Emplear métodos de accesibilidad formales y de calidad.

Estado de conformidad

Las Pautas de Accesibilidad para el Contenido Web (WCAG) define los requerimientos para diseñadores y desarrolladores para mejorar la accesibilidad para personas con discapacidad. Las pautas definen tres niveles de conformidad: Nivel A, Nivel AA, y Nivel AAA. Vegapp cumple parcialmente con las pautas WCAG 2.0 nivel AA. Cumple parcialmente significa que algunas partes del contenido no están adaptadas completamente a los estándares de accesibilidad.

Compatibilidad con navegadores y asistencias tecnológicas

Vegapp está diseñada para ser compatible con las siguientes asistencias tecnológicas:

  • Navegador web Mozilla Firefox con lector de pantalla NVDA en sistema operativo Windows 10
  • Navegador Móvil Google Chrome con lector de pantalla TalkBack en sistema operativo Android 6.0.1

Especificaciones técnicas

Accessibility of Vegapp está basado en las siguientes tecnologías para trabajar con una combinación particular de navegador web y cualquier asistencia tecnológica o plugins instalados en tu computadora:

  • HTML
  • WAI-ARIA
  • CSS

Se basa en estas tecnologías para cumplir con los estándares de accesibilidad utilizados.

Limitaciones y alternativas

A pesar de nuestros mejores esfuerzos para garantizar la accesibilidad de Vegapp, es posible que existan algunas limitaciones. A continuación se muestra una descripción de las limitaciones conocidas y las posibles soluciones. Comuníquese con nosotros si observa un problema que no se enumera a continuación.

Limitaciones conocidas de Vegapp:

  1. Aplicación en su totalidad: No funciona en sentido horizontal porque la página no fue maquetada para ser utilizada en esta orientación. Nos estamos capacitando para tener más conocimientos tecnológicos para poder hacer una web adaptable. .
  2. Detalles de receta: Algunas conceptos gastronómicos están en otro idioma y es posible que no sea interpretado por asistencias tecnológicas porque desde el código no está aclarado el cambio de idioma para estas asistencias. Debemos corregir el código para comunicarle a la asistencia tecnológica el cambio de idioma.
  3. Tamaño insuficiente en elemento "Volver" presente en todas las vistas y en elemento "Linterna" presente en Escáner.: El tamaño del elemento "Volver" y "Linterna" no tienen la medida mínima de 44x44 px. por lo tanto el área clickeable es chica. Debemos aumentar el tamaño en px de ambos elementos.
  4. Heading inadecuado en Recetas: Heading de nivel 2 no comunica visualmente la jerarquía que tiene por una mala elección de estilo a través de CSS. Debemos modificar el estilo a través de CSS. .
  5. Escaner, lista de recetas y detalle de recetas inadaptables: El contenido no se adapta correctamente al espaciado de texto porque la maquetación no responsive. Nos estamos capacitando para tener más conocimientos tecnológicos para poder hacer una web responsive.
